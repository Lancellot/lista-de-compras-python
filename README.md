# lista-de-compras-python

Aplicativo de linha de comando para gerenciar uma lista de compras. Desenvolvido em Python puro, sem dependências externas.

## Funcionalidades

- Adicionar produtos com nome, unidade de medida, quantidade e descrição
- Remover produtos pelo ID
- Pesquisar produtos por nome ou parte do nome
- Listagem automática dos produtos a cada iteração do menu
- ID único e automático para cada produto
- Tratamento de entradas inválidas

## Como executar

```bash
python lista_de_compras.py
```

## Menu de opções

```
1. Adicionar produto
2. Remover produto
3. Pesquisar produtos
4. Sair do programa
```

| # | Unidade     |
